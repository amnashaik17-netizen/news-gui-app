# 📰 News GUI App (Python)

## 📌 Description

This is a Python-based GUI application that fetches live news headlines from BBC News using web scraping.

## 🚀 Features

* Fetch latest news headlines
* Clickable links (opens in browser)
* Clean dark-themed GUI
* Built with Tkinter

## 🛠️ Technologies Used

* Python
* Tkinter
* Requests
* BeautifulSoup

## ▶️ How to Run

1. Install dependencies:
   pip install requests beautifulsoup4

2. Run the app:
   python news_gui.py

## 📂 Output

* Displays news in GUI window
* Click links to open full article
